# Pokébola Misteriosa 

Projeto web interativo onde o usuário clica em uma pokébola para descobrir qual Pokémon foi sorteado.

A aplicação simula uma mecânica simples de sorteio, com animação da pokébola e revelação aleatória de um Pokémon.

##  Objetivo

Este projeto foi desenvolvido como atividade prática com foco em:

- Desenvolvimento web com *HTML, CSS e JavaScript*
- Manipulação do *DOM*
- Implementação de *responsividade para dispositivos móveis*
- Publicação de projetos utilizando *GitHub Pages*

##  Funcionalidades

- Clique na pokébola para iniciar o sorteio
- Animação da pokébola
- Sorteio aleatório de Pokémon
- Interface responsiva para mobile

##  Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git e GitHub
